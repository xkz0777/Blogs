## 异常处理

在 C 语言中，对于异常的处理，只能使用 `if` 或 `switch` 等语句，判断错误码（多为 `int` 类型），进行对应的操作，而在 Java 等高级语言中，都从语言层面上提供了异常处理机制

## 注释和注解

### 注释

Java 中共有 3 种注释方式，`//` 和 `/* */` 的方式与 C++ 相同，还有一种 javadoc 工具采用的 doc 注释，如：

```java
/**
 * @author xkz
 * @version 1.0
 */
```
